# ContagemRegressiva
Resposta do exercício "ContagemRegressiva"


**Enunciado**

Faça um programa que exiba uma contagem regressiva no seguinte formato:

Preparar...
Apontar...
FOGO!

**Como executar**

Abra a pasta do programa e digite:

´´´

dotnet run
´´´

Pressione a tecla ENTER para cada transição.
